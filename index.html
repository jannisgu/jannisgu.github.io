Automated Stock Calculator

Keypoints:

  • Fullstack Application
  • Webscraping
  • User Authentication
Frontend
Backend

This is a personal project which I started with the intention of comparing stocks efficiently and based on an algorithm. The calculation is based on the discounted cashflow method. The application covers user authentication and allows users to evaluate a given stock within a few seconds automatically based on the financial data that can be found on Yahoo Finance. The application also provides an adaptable comparison of all calculated stocks as well as an evaluation page which is used to determine whether or not the calculations are effective or not.
